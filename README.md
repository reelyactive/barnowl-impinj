barnowl-impinj
==============

Collect ambient RAIN RFID data from Impinj readers.


Installation
------------

    npm install barnowl-impinj


Quick Start
-----------

Clone this repository, install package dependencies with `npm install`, and then from the root folder run at any time:

    npm start
